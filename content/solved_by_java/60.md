* 递归：左边`down`右边`up`

```java
import java.util.*;


public class Main {

    public static Scanner sc = new Scanner(System.in);

    static void rev(int i, int n, boolean isDown){
        if(i > n){
            return;
        }
        rev(i + 1, n, true);
        if(isDown){
            System.out.println("down");
        }else {
            System.out.println("up");
        }
        rev(i + 1, n, false);
    }

    public static void main(String[] args) {
        int n = sc.nextInt();
        rev(1, n, true);
    }

}
/*
     down
  down   up
down up down up

 */
```
